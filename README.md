# Expense Core TypeScript

A small TypeScript backend logic project for learning clean entity design, validation, encapsulation, and business methods.

## What I learned

- TypeScript literal union types
- `as const`
- Deriving types from arrays
- Classes
- Private fields
- Readonly fields
- Getters
- Business methods
- Constructor validation

## Current Features

- Expense entity
- Valid expense categories
- Title validation
- Amount validation
- Mark expense as paid/unpaid
- Update title, amount, and category safely
